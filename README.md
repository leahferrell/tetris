# Tetris

[Work in Progress]

Tetris clone written with React and Redux.

## About

Developed by Leah Ferrell

## How to Run

1. Clone this repository to your local machine
2. `npm install`
3. `npm start`
4. Navigate to: `http://localhost:3000`
